EquaSolve is a C++ program that can solve first-degree equations that 
follow a specific syntax (for now).

It converts string equations (e.g., 10 + x = 5) into a character array and
uses custom functions that I have developed to solve the equation.

Acceptable syntaxe :
	- 10+x=100
	- 12+x = 3
Inacceptable syntaxe:
	- x+10=13
	- 45 + x = 7
	-2 = 56+x

First version , wille be better over time
