EquaSolve is a C++ program that can solve first-degree equations .

It converts string equations (e.g., 10 + x = 5) into a character array and
uses custom functions that I have developed to solve the equation.

Now support negative value but need at least two integer and a x 

Syntaxe exemple :
-5=5+x
456200+x=-45
-789484564564+x=-69
